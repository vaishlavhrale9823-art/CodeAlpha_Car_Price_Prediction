# Car Price Prediction 

This project is part of CodeAlpha Data Science Internship.

##  Objective
To predict the price of used cars using machine learning techniques.

##  Dataset
The dataset contains information about cars such as:
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

##  Model Used
Linear Regression

##  How to Run
1. Install required libraries:
   pip install pandas numpy scikit-learn matplotlib

2. Run the script:
   python car_price_prediction.py

##  Output
The model predicts the selling price of a car based on input features.
